# Папка для изображений

Поместите сюда все изображения для вашего сайта.

## Необходимые файлы:

### Главный экран
- `hero-bg.jpg` - Фоновое изображение для главного экрана
  - Рекомендуемый размер: 1920x1080px или больше
  - Может быть постер фильма или атмосферный кадр

### Локации (3+ изображения)
- `location1.jpg` - Первая локация
- `location2.jpg` - Вторая локация
- `location3.jpg` - Третья локация
  - Рекомендуемый размер: 800x600px или больше
  - Добавьте больше файлов если нужно

### Костюмы (3+ изображения)
- `costume1.jpg` - Первый костюм
- `costume2.jpg` - Второй костюм
- `costume3.jpg` - Третий костюм
  - Рекомендуемый размер: 600x800px или больше

### Художественно-постановочная часть (3+ изображения)
- `art1.jpg` - Мудборд / цветовая палитра
- `art2.jpg` - Референсы
- `art3.jpg` - Атмосфера / концепция
  - Рекомендуемый размер: 800x600px или больше

## Форматы изображений

- JPG/JPEG - для фотографий
- PNG - для изображений с прозрачностью
- WebP - для лучшего сжатия (современный формат)

## Советы

- Используйте качественные изображения высокого разрешения
- Оптимизируйте размер файлов (не более 500KB на изображение)
- Сохраняйте соотношение сторон изображений
- Можете добавить больше изображений и обновить HTML соответственно
