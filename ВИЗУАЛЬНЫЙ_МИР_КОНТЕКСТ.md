## Техническая реализация

- Каждая локация/реквизит/костюм = карточка с превью + клик открывает слайдер
- Слайдеры хранятся в объекте `locationSliders` в script.js
- Папки с изображениями в `/images/` с названием локации
- CSS: object-position: top для всех карточек (styles.css)
- Для Саши индивидуальный inline style: object-position: center 30%

## Как добавить новую локацию/реквизит/костюм

1. Создать папку в `/images/НазваниеПапки/`
2. Добавить изображения
3. Добавить запись в `locationSliders` в script.js
4. Добавить карточку в HTML (соответствующий блок в секции "Визуальный мир")
5. Обновить версии CSS и JS в index.html
